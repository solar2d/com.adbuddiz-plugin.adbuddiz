## Overview

Returns `true` if the next call to [adbuddiz.showAd()][plugin.adbuddiz.showAd] will successfully display an ad.

## Syntax

	adbuddiz.isReadyToShowAd( [placement] )

##### placement ~^(optional)^~
_[String][api.type.String]._ The placement name.
