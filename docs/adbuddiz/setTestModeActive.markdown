# adbuddiz.setTestModeActive()

> --------------------- ------------------------------------------------------------------------------------------
> __Type__              [Function][api.type.function]

## Overview

Activates test mode (will display test ads).

Call before [adbuddiz.cacheAds()][plugin.adbuddiz.cacheAds].

## Syntax

	adbuddiz.setTestModeActive()
